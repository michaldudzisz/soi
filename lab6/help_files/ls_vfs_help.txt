Usage: ls_vfs [OPTION]... VFS_DISC

List informations about VFS_DISC (which is a filename). With no options given, prgoram lists all filenames stored in specified virtual file system. Available options:
	-i 		do memory map printing informations about each inode
	-l 		list files with their sizes

Warning: option -i is stronger than -l and in case of passing both -i and -l program will be run as with -i option only.
